About JavaScript, formally called EcmaScript

  • Initially a small scripting language created for the web browser, released in Netscape navigator.
  • Very easy to learn but many mistakes in the language mean you have to be careful and know about the dark corners.
  • Based on a prototype model, very different from other languages, you can't think of it like regular object-oriented coding.
  • Now the world's most popular language, probably the most important language to know now.
  • Incredibly flexible language that allows different styles but convention, style has emerged.
  • Introduction of nodejs brought JavaScript to the server side which in turn brought an explosion of JavaScript frameworks and code.
  • Introduction of ReactJS, by my friend Jordan Walke at Facebook, further increased the importance of JavaScript, (ReactNative makes it even more important)

Basic factorial example 

	    
let factorial = function (num) {
    // If the number is less than 0, reject it.
    if (num < 0) {
        return -1;
    }
    // If the number is 0, its factorial is 1.
    else if (num == 0) {
        return 1;
    }
    let tmp = num;
    while (num-- > 2) {
        tmp *= num;
    }
    return tmp;
}
let result = factorial(8);
